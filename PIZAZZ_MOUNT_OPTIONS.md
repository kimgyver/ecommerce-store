# Pizazz Edge Mounts - 3가지 옵션 설명

## 마운팅 시스템의 구조

```
Wall (벽)
  ↓
G부품 (벽에 고정)
  ↓
L부품 + H부품 (G부품에 슬라이드/고정)
  ↓
I부품 (나사로 G부품에 고정)
```

---

## Option 1: 투명 패널 사이 디스플레이

```
┌─────────────────────────┐
│  Back Panel (뒷배경)     │  ← 색상: 검은색, 금색 등
├─────────────────────────┤
│                         │
│   YOUR IMAGE HERE       │  ← 고객의 이미지
│                         │
├─────────────────────────┤
│  Clear Front Panel      │  ← 투명 패널 (이미지 보호)
└─────────────────────────┘

L부품으로 고정
```

**특징:**

- 이미지가 뒷배경과 투명 앞판 사이에 "끼워져" 있음
- 고급스럽고 세련된 느낌
- 뒷배경 색상으로 분위기 결정

---

## Option 2: 프린팅된 패널 디스플레이

```
┌─────────────────────────┐
│   Back Panel (선택사항)  │  ← 있을 수도, 없을 수도
├─────────────────────────┤
│                         │
│  PRINTED IMAGE PANEL    │  ← 이미지를 패널에 직접 프린팅
│  (이미지가 패널에 인쇄) │
│                         │
└─────────────────────────┘

L부품으로 고정
```

**특징:**

- 이미지를 직접 패널에 프린팅
- Option 1보다 간단하고 저렴할 수 있음
- 뒷배경 없이 프린팅 패널만 사용 가능

---

## Option 3: 보더 기능 (가장 특별함! ⭐)

```
┌───────────────────────────────┐
│   Back Panel (큰 사이즈)       │
│  ┌───────────────────────┐    │
│  │                       │    │
│  │   YOUR IMAGE HERE     │    │  ← 이미지 주변에 보더 생김
│  │                       │    │
│  └───────────────────────┘    │
│   ← 뒷패널이 이미지보다 큼    │
└───────────────────────────────┘

L부품으로 고정
```

**특징:**

- 뒷패널을 이미지보다 **더 크게** 사용
- 이미지 주변에 자동으로 "프레임" 같은 보더 생성
- 시각적 임팩트 대폭 증대
- **경쟁사들은 이것을 못 함! (Pizazz만의 강점)**

## 실제 예시

### Option 1 예시: 산 풍경 사진

```
검은색 뒷패널
  ↓
[        🏔️ 산 사진 🏔️        ]  ← 투명판과 뒷판 사이
  ↓
투명 앞판
```

### Option 3 예시: 같은 사진에 보더

```
큰 검은색 뒷패널
  ↓
[              ]
[   🏔️ 산 사진 🏔️   ]  ← 주변에 검은색 보더 프레임
[              ]
```

**차이:** 보더가 있으면 더 멋있고 전문적으로 보임!

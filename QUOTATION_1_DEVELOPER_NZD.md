## 1️⃣ 뉴질랜드 시장에서 왜 NZ$100/시간인가?

### 뉴질랜드 풀스택 개발자 시장 요율

```
Auckland/Wellington (대도시):
┌─ 주니어 (0-3년): NZ$70-85/시간
├─ 미드레벨 (3-7년): NZ$90-120/시간 ⭐ 대부분이 여기
├─ 시니어 (7년+): NZ$120-160/시간
└─ 매우 경험 많은: NZ$160-200/시간

에이전시:
├─ 소규모: NZ$150-200/시간
├─ 중규모: NZ$200-300/시간
└─ 대규모: NZ$300-500/시간
```

### 우리 제안의 위치

```
NZ$100/시간 = 경험 있는 개발자의 합리적 가격

✅ 이유:
- 시장의 "중상" 수준 (경험 충분)
- 에이전시보다 40-50% 저렴
- 풀스택 역량 보유
- 프로젝트 복잡도에 적합
- 뉴질랜드 생활비 반영
```

### vs. 다른 옵션

```
NZ$70/시간:
- 경험 부족한 개발자
- 품질 위험 (이전처럼 실패 가능)
- 예상 기간: 14-16주

NZ$100/시간: ⭐ 우리 제안
- 경험 충분한 개발자
- 안정적인 품질
- 예상 기간: 13주
- 가성비 최고

NZ$120-150/시간:
- 매우 경험 많은 개발자
- 불필요하게 비쌈 (+20-50%)
- 예상 기간: 13주 (같음)

에이전시 (NZ$200+/시간):
- 마진이 크게 포함됨
- 예상 기간: 10-12주 (병렬 가능하지만 1명이므로 불가)
- 비용 3배 이상
```

```

├─ Frontend 작업: 100시간

├─ DevOps/배포: 25시간

1명이 모든 것을 하므로 오버헤드:

- 예상 밖의 버그: +15%

실제 필요 시간: 305 × 1.45 = 442시간 ≈ 440시간
```

### 가격 계산

```
필요 시간: 440시간
시간당 요율: NZ$100/시간
─────────────────
```

## 7️⃣ 1명 개발자로 실제 가능한가?

### 가능한 이유

```
✅ 요구사항이 완벽하게 정의됨
   - 우리가 함께 이미 정의함 (NEW_WEBSITE_REQUIREMENTS.md)
   - 추가 변경사항 최소화 가능
   - 명확한 범위 제한

✅ 신뢰할 수 있는 기술 스택
   - Next.js (성숙한 프레임워크)
   - PostgreSQL (안정적인 DB)
   - Prisma (ORM, 생산성 높음)
   - 많은 레퍼런스 있음

✅ "신규" 개발 (레거시 X)
   - 기존 코드 정리 불필요
   - 아키텍처 자유로움
   - 기술 선택 유연함

✅ 데이터 마이그레이션 불필요
   - Phase 1부터 새로운 DB로 시작
   - 기존 사이트는 병행 운영 가능
```

### 위험 요소 및 완화

```
⚠️ 위험: 한 사람이 병목
✅ 완화: 주 1회 진행 상황 보고 + 4주마다 마일스톤 검토

⚠️ 위험: 휴가/아픔 시 중단
✅ 완화: 문서화 충실 + 긴급 연락 체계 구축

⚠️ 위험: 테스트/QA 시간 부족
✅ 완화: 자동화 테스트 + 배포 전 귀사 UAT

⚠️ 위험: 배포/DevOps 경험 부족
✅ 완화: Vercel 사용 (배포 자동화) + AWS 프리티어
```
